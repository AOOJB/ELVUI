# ELVUI Mod
Includes: <br>
current health with abssorbs tag(current health + absorbs in the same number). Text format: [health:current-with-absorbs] <br>
current health deficit with absorbs tag(Shows ur current health + absorbs relative to ur max hp). Text format: [health:currentdeficit-with-absorbs] <br>

Download Tags.lua. <br>
Go to "AddOns\ElvUI\Core\General" and replace Tags.lua. <br>
